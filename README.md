# tpl-aws-website

> [!IMPORTANT]
> Template monorepo with a AWS-hosted static website infrastructure and automations.

## Monorepo sub-projects

- `fe/` - Frontend
- `infra/` - Infrastructure

## Usage

```
$ make
'''
Available targets:
  deploy - build & deploy infra and frontend
  fe-% - fe targets
  infra-% - infra targets
'''
```
